About Django

Django is a Python-based free and open-source web framework that follows the model–template–views (MTV) architectural pattern.

What is MVT?

Model: definitive source of information about your data.

Template: is a text files, It can generate any text-based format (HTML, XML, CSV, etc.).

Views: Python functions or classes that receive a web request and return a web response.
